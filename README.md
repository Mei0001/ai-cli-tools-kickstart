# AI CLI Tools Kickstart Guide

今生成AI界隈で話題のAI CLIツール（Claude Code, Codex, Gemini CLI, Cursor CLI, GitHub Copilot CLIなど）の内容について初学者向けに説明、テスト、比較などが可能なキックオフ資料を作成するプロジェクトです。

## 対象ツール

- Claude Code
- Codex
- Gemini CLI
- Cursor CLI
- GitHub Copilot CLI
- その他のAI CLIツール

## 目的

- 各AI CLIツールの初学者向け説明
- 各ツールのテストと比較
- キックオフ資料の作成

## 構成

このリポジトリでは、各AI CLIツールについて以下の情報をまとめていきます：

- 概要・特徴
- インストール方法
- 基本的な使い方
- 使用例・デモ
- 比較表
- 参考資料
